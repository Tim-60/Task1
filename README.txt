При переводе тернарного оператора в if...else и switch() для уменьшения объема кода некоторые заданные в тернарном операторе условия были убраны из-за математической невозможности их выполнения или автоматического выполнения соответственно. 
Результат перевода совпал с результатом, возвращенным тернарным оператором.
